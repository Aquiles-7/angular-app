1. Introducción

Este documento presenta la documentación técnica completa de la aplicación "Catálogo Tienda de Productos", un proyecto desarrollado como trabajo final de la materia Programación 4. Se trata de una aplicación web moderna que permite a los usuarios navegar, buscar y comprar productos de una tienda en línea.

La aplicación ha sido construida utilizando Angular 21, un framework moderno y robusto para el desarrollo de aplicaciones web. Incluye funcionalidades de catálogo de productos, sistema de carrito de compras, búsqueda avanzada, integración con WhatsApp para notificaciones de pedidos, y un panel de administración para gestionar productos.

2. Descripción General del Proyecto

"Catálogo Tienda de Productos" es una aplicación web que simula el funcionamiento de una tienda en línea. Los usuarios pueden:

  • Explorar categorías de productos
  • Ver detalles de productos individuales
  • Buscar productos por nombre o categoría
  • Agregar productos al carrito de compras
  • Modificar cantidades de productos en el carrito
  • Proceder con el checkout
  • Recibir confirmación de pedidos por WhatsApp

Los administradores pueden acceder a un panel especial donde pueden:

  • Crear nuevos productos
  • Editar productos existentes
  • Eliminar productos del catálogo
  • Gestionar categorías de productos

7. Servicios

7.1 ProductService

Gestiona todas las operaciones relacionadas con productos. Utiliza HttpClient para obtener datos de un archivo JSON y RxJS BehaviorSubject para manejar el estado de los productos.

Métodos principales:
  • getAll(): Obtiene todos los productos
  • getProducts(): Retorna observable de productos
  • getProductByCategory(category): Filtra por categoría
  • getProductById(id): Obtiene un producto específico
  • searchProducts(parameter): Busca productos por texto o categoría

7.2 CartService

Gestiona el carrito de compras del usuario. Los datos del carrito se persisten en localStorage para mantener el estado entre sesiones.

Métodos principales:
  • addItem(product): Agrega un producto al carrito
  • getItems(): Obtiene todos los items del carrito
  • deleteItem(productId): Elimina un producto del carrito
  • changeQuantity(productId, quantity): Modifica la cantidad
  • updateStorage(): Sincroniza carrito con localStorage

7.3 CategoryService

Gestiona las categorías de productos disponibles en la aplicación.

7.4 WhatsAppService

Integra WhatsApp Web para enviar notificaciones de pedidos confirmados a los clientes. Utiliza la API de WhatsApp para generar mensajes con detalles del pedido.
